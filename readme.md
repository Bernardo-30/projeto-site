## 💻 Projeto

O DevLinks é um agregador de links para usar como um cartão de visitas online.

## ✨ Tecnologia

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e GitHub
- Figma
